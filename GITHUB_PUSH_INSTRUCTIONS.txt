TROUBLESHOOTING
================================================================================

Error: "Permission denied (publickey)"
--------------------------------------
Solution: Use HTTPS URL instead of SSH
git remote set-url origin https://github.com/Sai-sathvik09/menuflow.git

Error: "Repository already exists"
-----------------------------------
Solution: The repo has been created, just push:
git push -u origin main

Error: "Authentication failed"
-------------------------------
Solution: Use a Personal Access Token instead of password
See Step 3 above for creating a token

Error: "Unable to push to git"
-------------------------------
Solution: Try the download and manual upload method (Option 3)
